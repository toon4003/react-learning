# react-learning
for learning react

## Installation
Run this cmd

```bash
npm create vite@latest <project_name>
```

It will ask:

```
Select a framework:
❯ Vanilla
  Vue
  React
  Preact
  Lit
  Svelte
```

Choose:

```
React
```

Then it asks:

```
Select a variant:
❯ JavaScript
  TypeScript
```

Choose:

```
JavaScript
```

---

### Then install dependencies

```bash
cd <project_name>
npm install
npm run dev
```

Now Project runs at:

```
http://localhost:5173
```
